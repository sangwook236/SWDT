	- dotnet CLI.
		https://learn.microsoft.com/en-us/dotnet/core/tools/
		https://learn.microsoft.com/en-us/nuget/consume-packages/install-use-packages-dotnet-cli

		The CLI tool for .NET Core and .NET Standard libraries, and for any SDK-style project such as one that targets the .NET Framework.
