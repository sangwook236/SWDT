[-] Tools.
	https://clang.llvm.org/extra/

	- clang & clang++.
		https://clang.llvm.org/

	- LLDB (debugging tool).
		https://lldb.llvm.org/

	- clang-tidy (static code analyzsis tool).
		https://clang.llvm.org/extra/clang-tidy/
		https://clang.llvm.org/extra/clang-tidy/checks/list.html

		./.clang-tidy

		clang-tidy -checks=cppcoreguidelines-* <FILE>
		clang-tidy -checks=cppcoreguidelines-*,clang-analyzer-* <FILE>
			If you're using clang-tidy, you can naturally make the static analyzer run alongside clang-tidy by enabling the clang-analyzer checks.
		clang-tidy -checks=cppcoreguidelines-*,clang-analyzer-*,modernize-* <FILE>
		clang-tidy -checks=cppcoreguidelines-*,clang-analyzer-*,modernize-use-nullptr,google-readability-casting <FILE>
		clang-tidy -checks=cppcoreguidelines-*,clang-analyzer-* -system-headers <FILE>

	- Clang Static Analyzer (static code analyzsis tool).
		https://clang-analyzer.llvm.org/

		clang --analyze <FILE>
			Output to <FILE>.plist.
		clang --analyze -o <OUTPUT_FILE> <FILE>
		clang --analyze -Xclang -analyzer-output=html <FILE>

	- Sanitizer.
		https://clang.llvm.org/docs/

		AddressSanitizer:
			https://clang.llvm.org/docs/AddressSanitizer.html
			clang++ -O1 -g -fsanitize=address -fno-omit-frame-pointer <FILE>
		LeakSanitizer:
			https://clang.llvm.org/docs/LeakSanitizer.html
			clang -fsanitize=address -g <FILE> ; ASAN_OPTIONS=detect_leaks=1 <EXECUTABLE>
		MemorySanitizer:
			https://clang.llvm.org/docs/MemorySanitizer.html
			clang -fsanitize=memory -fno-omit-frame-pointer -g -O2 <FILE>

		TypeSanitizer:
			https://clang.llvm.org/docs/TypeSanitizer.html
		DataFlowSanitizer:
			https://clang.llvm.org/docs/DataFlowSanitizer.html
		ThreadSanitizer:
			https://clang.llvm.org/docs/ThreadSanitizer.html
		UndefinedBehaviorSanitizer:
			https://clang.llvm.org/docs/UndefinedBehaviorSanitizer.html

	- Clang Format.
		https://clang.llvm.org/docs/ClangFormat.html
		https://clang.llvm.org/docs/ClangFormatStyleOptions.html

		./.clang-format

		clang-format -i --style=file <FILE>

	- clangd.
		https://clangd.llvm.org/

		A language server that can work with many editors via a plugin.
