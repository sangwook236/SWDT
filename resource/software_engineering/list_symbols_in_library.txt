[-] Static library.
	- Windows.
		Use 7-zip.
			Exported symbols are listed in a txt file.
		lib /list <LIBRARY>.lib | grep <FUNCTION>
		dumpbin /archivemembers <LIBRARY>.lib | grep <FUNCTION>
	- Linux.
		nm <LIBRARY>.a | grep <FUNCTION>
		objdump --syms <LIBRARY>.a

[-] Shared library.
	- Windows.
		Use Dependency Walker.
		dumpbin /exports <LIBRARY>.dll | grep <FUNCTION>
	- Linux.
		nm -D <LIBRARY>.so | grep <FUNCTION>
		readelf --syms <LIBRARY>.so | grep <FUNCTION>
		objdump --dynamic-syms <LIBRARY>.so

[-] Import library.
	- Windows.
		dumpbin /exports <LIBRARY>.lib | grep <FUNCTION>
		dumpbin /archivemembers <LIBRARY>.lib | grep <FUNCTION>
		Use 7-zip.
			Shared library file names list up.
			Exported symbols list up in a txt file.
