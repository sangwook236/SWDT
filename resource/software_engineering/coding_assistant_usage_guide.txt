	- Usage.
		Inline Chat: Ctrl+I

		Comands:
			/dev
				Generates new code across your entire project and implement features.
			/doc
				Writes API, technical design, and onboarding documentation.
			/review
				Performs code reviews, flagging suspicious code patterns and assessing deployment risk.
			/test
				Generates unit tests and adds them to your project, helping you improve code quality, fast.
			/transform
				Upgrades your Java applications in minutes, not weeks.
			/help
			/clear

		@workspace
			Add @workspace to the beginning of your message to include your entire workspace as context.
